El proposito del sitio

Este sitio esta hecho on la intencion de dar a conocer conceptos basicos pero practicos de CSS que permitiran al estudiante en HTML una mejor comprencion de etiquetas, propiedades, y demas caracteristicas para poder mejorar sus sitios web.

Tema 1: Sintaxis y Selectores en CSS

La sintaxis CSS refleja este objetivo y sus bloques de construcción básicos son:
+ La propiedad: define la caracteristica que se considera
+ El valor: que difine el modo de actuar de la caracteristica o propiedad, identificado por el motor de busquedad.

MDN Contribuciones (2019).Sintaxis CSS.


Los SELECTORES en CSS definen sobre qué elementos se aplicará un conjunto de reglas CSS. Existen muchos tipos de selectores como los Basicos, los combinados y mas complejos selectores de pseudoclases y pseudoelementos
